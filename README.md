charts
======

- 可视化数据图表，发挥前端之优势，收集各种在网页优雅展示数据的具体方案。

- 图表类型：
  - 条形图
  - 柱状图
  - 折线图
  - 饼图
  - 散点图
  - 面积图
  - 圆环图
  - 雷达图
  - 气泡图
  - 股价图
  - 复合图（图表间的相互叠加来形成）

- 第三方网站提供他们认为的图表解决方案：
  - [highcharts](http://www.highcharts.com/)
  - [jqplot](http://www.jqplot.com/deploy/dist/examples/)
  - [D3](http://d3js.org/)
  - [chartjs](http://chartjs.devexpress.com/)
